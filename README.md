# 📝 MyTodo App

MyTodo is a simple and user-friendly todo list application to help you manage daily tasks efficiently.

## ✨ Features
- Add new task
- Edit task
- Delete task
- View deleted task history
- Restore deleted tasks
- Permanently delete tasks

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript

## 🚀 How to Use
1. Type your task in the input field
2. Click "Add"
3. Manage your tasks easily

## 📁 Project Structure
- index.html
- style.css
- script.js
